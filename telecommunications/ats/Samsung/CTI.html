CTI приложения

 

 

OfficeServ Call

Ключевые характеристики:
Папка контакты, вызов абонента, управление аппаратом.
Создание различных записных книжек, работа с корпоративной записной книжкой.
Функция перетащи и вбрось, индикаторные клавиши.
Управление функциями телефона, задачник.
Запись информации о вызовах.
 

OfficeServ EasySet

Ключевые характеристики:
Установка персонального пароля, блокировка аппарата.
Задание имени аппарата
установка переадрессаций, текущего времени и даты, персональный список быстрого набора.
Будильник, Программируемые сообщения об отсутствии.
Отправка сообщений, учет разговоров.
 

OfficeServ Link

Программа OfficeServ Link предназначена для контроля лицензий подключения CTI приложений Samsung.
 

OfficeServ OpenTSP

Драйвер для взаимодействия TAPI приложениями через Open TSP интерфейс.

 
  OfficeServ Operator

 

 Данное приложение позволяет удобным образом организовать рабочее место оператора.
  OfficeServ DataView

 

OfficeServ DataView имеет следующие компоненты: модуль сбора данных, который анализирует системные события, модуль обработки данных, который вычисляет статистику по полученным данным и сохраняет статистические данные в базе данных, и модуль сообщений, который предоставляет информацию в виде WEB страницы. Модули называют Сборщиком Данных, Менеджером Данных, и Обозревателем соответственно.
  OfficeServ SoftPhone

Программный IP телефон для систем OfficeServ для установки на персональном компьютере.
  OS SoftPhone (PPC)

 

Программный IP телефон для систем OfficeServ для установки на Pocket PC.
  OfficeServ E-Mail G/W

Ключевые характеристики:
E-mail шлюз карт SVMi
Доставка голосовых(WAV)сообщений и факсовых(TIFF)сообщений в почтовый ящик(и)пользователя голосовой почты.
Уведомление о голосовых и факсовых сообщениях в почтовый ящик(и)пользователя голосовой почты.
  OfficeServ EasySync

Ключевые характеристики:
Программа синхронизации телефона ITP-5112L с ПК.
Управление телефонной книгой телефона.
Экспорт/Импорт контактов из MS Outlook.
Загрузка звуков и картинок заставок.
Отпрвка и прием тестовых сообщений.
